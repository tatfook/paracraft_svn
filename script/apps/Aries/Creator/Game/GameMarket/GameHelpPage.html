新版创意空间指南
游戏模式：
E键:打开背包; WASD键行走. Q键: 扔掉手中的方块
长按鼠标左键不放:删除; 点击鼠标右键创建物体
X键: 与最近的NPC对话
F键：进入/退出飞行模式. Space上升, X下降
右键点击快捷栏中物品：可移动物品
双击并按住W键可加速奔跑。～键打开望远镜
鼠标滚轮切换物品, Ctrl+鼠标滚轮:摄影机距离
创造模式：
Ctrl+Z: 撤销上一步操作; Ctrl+Y: 反撤销; Ctrl+D:切换选择区
Shift+左键: 创造模式: 删除选择块周围3*3内的所有同类的块。
Shift+右键: 复制当前鼠标的块直到被拦截 （创建线）
Alt+左键: 用当前块替换手中的块
Alt+右键: 用当前块替换与鼠标所在的块（替换单个颜色的块）
Alt+Shift+右键: 用当前块替换与鼠标所在的块相连的所有块 （批量替换颜色）
Ctrl+左键: 进入选择多块AABB选择模式。 再次Ctrl+左键可以增加新的选择， 左键点击任意地方退出选择模式。Ctrl+左键拖动：区域选择
实数地表上长按左键不放手可选择地面
TAB: 角色向上一层; Shift+TAB: 角色向下一层
Ctrl+S: 快速保存; Ctrl+F12: 隐藏UI
左键选择人物或模型， Ctrl+左键（可以选择多块积木）
- 再次点击左键可以移动，Ctrl+左键可以复制多分
当选中多块积木时， shift+左键, 可以移动当前选中的所有积木到新的位置
鼠标中键瞬移. F3查看位置信息
Enter键然后在聊天框中输入下面指令
/save:存盘
/ring radius [thickness]: 创建环形
/circle radius: 创建圆形
/create [id|filename] [bx] [by] [bz] 创建名为[filename]的模型或ID的物品
/del: 删除当前选中的块
/del -below [radius:1-256]: 删除当前人物高度以下的虽有所有块
/del -mode [real|block]：更改删除模式 实数或积木
/terrain -r [radius:1-256]: 删除当前人物周围的实数地表
/terrain -repair [radius:1-256]: 恢复当前人物周围的实数地表
/tp x y z: 传送到指定3D位置 /tp x z: 传送到指定2D位置 /tp home 返回出生点
/goto x y z: 传送到指定块的位置. /home 回家 /sethome 设置出生点
/fps:第一人称
/sel [radius] [height]:选择人物下方的方块
/replace [-all] [from_id] [to_id] [radius]: 批量替换块. radius默认为256.
/fill [block_id]: 选中区域用制定的块填充
/where 当前tile的Pos
/hide 隐藏当前角色 /hide desktop 隐藏UI
/show 显示当前角色 /show desktop 显示UI
/day [minutes_in_a_day] 多少分钟为1天. 无参数表示无限长
/time [-1,1] 设置当前时间, 0或空位是中午,-1或1是晚上
 Enter键然后在聊天框中输入下面指令
/shader 2 切换到高级渲染 /shader 1 切换到普通渲染
/shadow [0,1]
/skin [filename or id] 更换主角贴图皮肤. 空是随机
/renderdist [10-200] 可视距离
/light [0,2] [0,2] [0,2] 设置发光块的颜色.
/text [-w] [-p] [string] 在下方显示电影文字 -w 用户必须等待 -p 用户点击鼠标才继续
/tip [-name] string 在中间显示广播Tip消息. name是名字
/addrule name value 增加游戏规则
/rule 显示游戏规则
/rule [reset|reload] 重置或加载游戏规则
/dostring any_script_in_sandbox 运行一行脚本
/take [block] 将物品拿在手中
/give [block] 将物品送给人物
/setblock x y z [block] [data] 创建或消除方块（绝对位置）
/setblock ~-1 ~1 ~ (~-1 ~2 ~) [block] [data] 创建或消除一组2*3*1的方块（相对位置）
/clearbag 清空人物的背包
/offsetworld offsetY 将整个世界纵向移动offsetY格子(慎用)
 /mode [game|editor] 锁定游戏模式，无法用Ctrl+G切换.
/gravity [@playername] [value|9.8] 设置重力加速度
/speedscale [value|1] 设置人物运动相对速度。1是默认速度
/viewbobbing [true|false] 摄影机摇摆
/clone [-update] from_x from_y from_z (dx dy dz) to to_x to_y to_z (dx dy dz) [where sameblock]
/density 1.2 设置人物的密度
/name [name#tooltip1#tooltip2] 给手中的物品起名字
/t seconds [/其他命令] 在seconds秒后执行后面的命令
/move [@playername] [x y z] 可以移动指定人物到指定位置
/velocity [add|set] [@playername] x [y] [z] 增加|设置动物或人物的速度
/speeddecay [@playername] [surface_decay] [air_decay] 设置动物或角色在地面和空气中的摩擦力
/facing [@playername] angle 可以设置人物方向
/lookat [@playername] [x y z] 让当前人物看某个人物或方向
/scaling [@playername] size 改变人物的大小
/tickrate [@playername] rate 改变人物的更新刷新率，默认20. 30或60都OK
/fov [fieldofview:1.57] [animSpeed] 可以改变摄影机的FieldOfView
/loadworld [worldname|url|filepath] 可以在世界中切换世界
/loadregion [x y z] [radius] 强制加载所在位置坐标的地形范围的东西
/rain [strength] 、 /rain [strength] 开启
/record 开启关闭视频输出
/end 终止电影片段 例如 /t 30 /end 30秒后停止播放.
/movieclip [-stop] 暂停当前的电影片段
 /music [filename|1~6] [from_seconds] 播放指定ID的背景音乐或文件
/loadtemplate [x y z] templatename
/loadtemplate -a [seconds] [templatename] 模拟人创建场景的过程
/loadtemplate -r [templatename] 清空场景
/savetemplate [templatename] 保存模板，原点为Pivot
/durability [value] 设置目前手中装备的耐久度
/testblock x y z [(dx dy dz)] blockid [data] 用来判断目标或一个区域是否为指定方块
/compareblocks ~ ~-1 ~ (2 2 2) to ~5 ~ ~ 比较2个矩形区域中的方块是否完全一样
/anim [@playername] anim_name[,anim_name,...] 让人物做动作
/flood [radius or 10] [block_id] [x] [y] [z] 用水的方式填充人物所在区域
/unflood [radius or 10] [x] [y] [z] 吸干人物所在区域的水
/ellipsoid radiusX radiusY radiusZ beSolid 画椭圆体
/blockimage [-xy|-yz|-xz] [colors] filename [x y z] 将任意图片转换为方块
/sky [-tex] [sim|white|green|filename] 改变天空或天空的贴图.
/fog [-color|skycolor|fogstart|fogend] values 改变雾的颜色和距离
/torchcolor [r] [g] [b] 改变光源（火把）的颜色
/applytexturepack [folder_or_zipfile] 应用指定文件和目录的材质包。
/registeritem [block_id:2000-2999] texture [base_block_id] 创建新的物品类型。
例如: /registeritem 2000 Texture/blocks/lapis_ore.png 234
/hsv [h s v] [multiply_r multiply_g multiply_b] 调节整个屏幕的色相与饱和度
/grey [r g b] [glow_r glow_g glow_b] 开启全屏灰色效果
/viewport [@id] [alignment:_lt|_fi|_rt] [left] [top] [width] [height]改变主视图
/stereo [on|off] [eye_dist] 开启关闭3D立体输出模式
 /set -p name [=] prompt_msg 可以提示用户输入文字。
/set -p password=请输入密码:
/t ~1 /tip 你输入的密码是%password%
/call [code with return value] 执行任意代码. 例如:
/set mode=/call return GameLogic.GameMode:GetMode()
/if %mode%== "editor" /tip you are in editor mode
/if var1==var2 /othercommand 当var1==var2 时，运行后面的指令。命令支持>= > == <= < 操作符号
/jumpto [line_offset|end|begin] 跳转到指定位置的代码。
/if %name% == "0" /jumpto 3
/tip your name is NOT 0
/jumpto end
/tip your name is 0
/activate [x y z] 可以激活自己或其它方块。 可以用命令方块写GameLoop了。
/return [number|bool] 返回true
/pause 暂停当前人物或命令方块的所有/t指令的执行
/resume 继续当前人物或命令方块的所有/t指令的执行
/advancetime [~][time] 快进当前人物或命令方块的时间。 无参数表示快进到下一个关键帧。 ～表示相对时间否则是绝对时间。
/open [-p] url 可以打开URL地址. -p表示需要取得用户同意确认
/disableinput [@playername] [x y z] [true|false] 禁止命令方块，含羞草，或人物等Entity的所有输入. 在命令行中加入，可以产生让剧情对话只能看一遍的效果
 /startserver [ip_host] [port] 开启个人服务器
/connect [ip] [port] [username] [password] 链接到别人的服务器
/chat XXX 联网模式下可以聊天
/register username password 注册一个新用户: 用户数据在/config/ParaCraft/password.txt文件中, 管理员可自己添加。 如果删除这个文件为可自由加入.
/unregister username 删除一个用户
/configserver 打开服务器配置目录
