1. 宝石镶嵌有什么作用呢？
答：给装备镶嵌宝石是可以大幅度增加你的装备属性，使你在战斗的时候更强哦！
2. 什么样的装备可以镶嵌宝石呢？
答：有孔的装备是可以镶嵌宝石的哦，装备的孔有限，需要想清楚再镶嵌哦！
3. 同类宝石可以镶嵌多少个呢？
答：每种类型的宝石只能镶嵌一个，再次镶嵌同类宝石时，原有宝石会消失，直接镶上新的宝石哦！ 例如：已经镶嵌1级黄晶石，若镶嵌上5级黄晶石，成功后1级黄晶石的属性会直接替换成5级黄晶石的哦。
4. 什么是宝石配方？
答：配方是只合成高级宝石的方法，用的材料越多，合成率越高，5个宝石100%可以成功的！
5. 宝石合成失败了会怎么样呢？
答：宝石合成失败，用于镶嵌的材料全部都会消失，所以建议采用100%的合成配方哦！
6. 给装备镶嵌宝石失败了会怎么样呢？
答：宝石镶嵌失败，装备不会有变化，但用于镶嵌的宝石会降级或者消失，建议可以使用镶嵌符来增长成功率哦！
